
Book Scraper GUI (Version 2)

A Python web scraping application that extracts book data from BooksToScrape and exports filtered results to a CSV file through a user-friendly graphical interface.

⸻

Key Features
	•Clean and simple Tkinter GUI
	•User input validation (rating, price, pages)
	•Automatic price cleaning from website symbols
	•CSV export functionality
	•Page range control
	•No terminal interaction required

⸻

Tech Stack

Python • Requests • BeautifulSoup • Pandas • Tkinter

⸻

How to Run

pip install -r requirements.txt
python main.py

Enter the filtering values, click Start Scraping, and the CSV file will be generated automatically.
